# 🎬 Willy Movie App – Responsive Movie Browser

🚀 Live Demo: [movie-willy.vercel.app](https://movie-willy.vercel.app)

**Willy Movie App** is a modern and fully responsive movie browser built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **TMDB API**. The app features trending movies, new releases, a search page, and a dynamic hero section — all designed with clean, modern UI and mobile-first responsiveness.

---

## ✨ Features

- 🎥 Hero section with background image and title
- 🔥 Trending movies in a horizontal scroll layout
- 🆕 Infinite scroll for new releases
- 🔍 Search page with real-time movie filtering
- 📄 Movie detail page (optional for expansion)
- ⚡ Smooth transitions and responsive layout
- 📱 Optimized for mobile and desktop

---

## 🛠️ Tech Stack

| Tech              | Description                     |
| ----------------- | ------------------------------- |
| **Next.js**       | React framework with App Router |
| **TypeScript**    | Type-safe development           |
| **Tailwind CSS**  | Utility-first CSS styling       |
| **Framer Motion** | Smooth UI animations (optional) |
| **TMDB API**      | External movie data source      |
| **React Query**   | Data fetching & caching         |
| **Vercel**        | Deployment platform             |
